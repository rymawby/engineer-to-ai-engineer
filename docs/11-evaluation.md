# Evaluation

Evaluation tells you whether an AI system is improving, regressing, or simply sounding plausible.

## Concepts To Know

- Outcome evaluation
- Trajectory evaluation
- System metrics
- Golden datasets
- Recall@k
- LLM-as-judge
- A/B testing

## Revision Prompts

- What is the final outcome you want to measure?
- When does the path taken by the model matter?
- Which system metrics would you track?
- What belongs in a golden dataset?
- What does Recall@k measure in retrieval?
- What are the risks of using an LLM as a judge?
- When is A/B testing useful?

## Interview Angle

Evaluation should match the product risk. A search assistant, a support workflow, and a refund agent need different metrics and different levels of confidence.
