# Deterministic vs Agentic Systems

Production AI systems often combine deterministic software with model-driven reasoning. The design question is which part should own which responsibility.

## Concepts To Know

- Predictability vs flexibility
- Latency and cost trade-offs
- When agents make sense
- Why production systems avoid full agent loops

## Revision Prompts

- What makes deterministic workflows easier to operate?
- What flexibility do agentic workflows add?
- Why do agentic systems usually cost more and take longer?
- When is an agent loop justified?
- Why do many production systems constrain the model's choices?

## Interview Angle

Prefer deterministic systems for known workflows. Use agentic behaviour when the task requires dynamic reasoning, ambiguous planning, or flexible tool selection.
