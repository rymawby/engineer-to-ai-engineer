# Agents and Tool Use

Tool use lets a model request structured actions. Agent systems add loops, planning, state, and decision-making around those tool calls.

## Concepts To Know

- What tools are
- Function calling
- Tool use vs agents
- MCP servers
- Agent loops

## Revision Prompts

- What is a tool in an AI system?
- How does function calling make tool use safer than free-form text parsing?
- What is the difference between a model using a tool and an agent running a loop?
- What role can MCP servers play in a tool ecosystem?
- What can go wrong in an agent loop?

## Interview Angle

Tool use is often enough. Full agent loops should earn their place because they introduce cost, latency, uncertainty, and new failure modes.
