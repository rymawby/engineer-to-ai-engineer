# Senior-Level Thinking

Senior AI engineering is mostly judgement: knowing where models are useful, where ordinary software should stay in control, and how to operate the system safely.

## Concepts To Know

- Prefer deterministic systems first
- Use LLMs for reasoning, not truth
- Design for retries and failures
- Auditability
- Operational resilience

## Revision Prompts

- Which parts of the system can be deterministic?
- Where is the LLM adding reasoning or language ability?
- What can fail, and what happens next?
- What needs to be auditable later?
- How does the system degrade when a model, vector DB, or external tool is unavailable?

## Interview Angle

Use LLMs for reasoning and language. Use systems for state, truth, money, and safety.
