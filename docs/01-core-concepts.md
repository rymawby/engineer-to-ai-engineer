# Core Concepts

These are the baseline ideas behind most AI engineering discussions. You should be able to explain them without hiding behind jargon.

## Concepts To Know

- What is an LLM
- What is an embedding
- Embedding model vs LLM
- Stochastic systems
- Vector spaces

## Revision Prompts

- What does an LLM do well, and what does it not guarantee?
- How is an embedding different from generated text?
- Why are LLM outputs described as stochastic?
- Why do vector spaces make semantic search possible?
- What parts of an AI system should rely on normal software engineering instead of model behaviour?

## Interview Angle

A strong answer separates language generation from truth, state, and business rules. Treat the model as one component in a system, not as the whole system.
